# Follow this Instructions carefully      
1-Clone Repo
```
        $ git clone https://github.com/AhmadGadallahMostafa/Flicker-FrontEnd.git
```
2-Change to project dir.
```
    $ cd Flicker-FrontEnd
```
3- Download All dependencies
```
    $ npm install
```
4- Run In dev 
```
$ npm run json:server
$ npm run dev
```
